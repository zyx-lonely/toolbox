# PC Toolbox - 电脑工具箱

一款基于 Wails (Go + Vue 3) 开发的 Windows 系统工具箱，集成 80+ 实用功能。

## 功能模块

### 系统工具
- 系统信息 / 硬件检测报告 / 温度监控
- 进程管理器 / 电源方案 / 浏览器扩展
- 快捷键管理 / 系统激活 / 导出报告

### Windows 优化
- 磁盘清理 / 启动项管理 / 系统服务优化
- 系统健康体检 / 系统备份与还原 / 注册表清理
- 浏览器数据管理 / Windows 更新管理 / 环境变量管理
- 磁盘健康监控 / 服务依赖分析 / 文件关联管理
- **软件管理** - 基于 Windows Registry API 获取已安装软件列表

### 外部工具
- **外部工具管理** - 支持 ShellExecute runas 提权启动外部工具
- 系统必备软件

### 网络工具
- 网络诊断 / Hosts 编辑器 / IP 地理位置
- 局域网扫描 / WiFi 密码 / WiFi 扫描
- IP 冲突检测 / 局域网拓扑 / 批量 Ping
- 流量图表 / 远程桌面 / 端口占用查询
- DNS 切换 / IP 子网计算器 / 网速测试

### 文件工具
- 重复文件查找 / 大文件查找 / **文件预览**
- 图片格式转换 / 图片批量压缩 / 文本编码转换
- 文件哈希校验 / 文件内容搜索 / 文件内容替换
- 文件夹大小分析 / 文件差异对比 / PDF 工具
- 批量重命名 / 文件批量归类 / 文件解锁器
- 文件时间戳修改 / 批量正则重命名

### 开发工具
- JSON 格式化 / 编解码工具 / 代码美化
- HTTP 调试 / 正则测试 / JWT 解码
- YAML/TOML / UUID 生成 / Markdown 预览
- 二维码生成 / Cron 生成器 / 文本对比
- 文本字数统计 / JSON/CSV 互转

### 安全工具
- 文件加密解密 / 密码生成器 / 文件粉碎

### 日常工具
- 单位换算 / 取色器 / 截屏工具
- 番茄时钟 / **剪贴板历史（支持图片）** / 定时任务
- 便签 / 提醒事项 / **屏幕标尺**

### 设置
- 应用设置 / 右键菜单 / 检查更新 / 主题商店 / 操作日志

## 技术栈

- **后端**: Go + Wails v2
- **前端**: Vue 3 + TypeScript + Naive UI + Pinia
- **构建**: Vite

## 开发

```bash
# 安装依赖
cd frontend && npm install

# 开发模式
wails dev

# 生产构建
wails build
```

## 构建产物

构建完成后，可执行文件位于 `build/bin/pc-toolbox.exe`。
