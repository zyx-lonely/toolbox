# PC Toolbox v1.5.0 发布说明

## 🎉 新版本发布

**版本号**: v1.5.0  
**发布日期**: 2026-06-22  
**构建日期**: 20260622

## ✨ 主要变更

### 1. 移除系统监控功能
- 移除不准确的系统监控仪表板
- 简化应用功能，专注于核心功能

### 2. 改进软件管理功能
- 使用更准确的方法获取已安装软件列表
- 使用 PowerShell 和 WMI 双重验证
- 支持显示安装位置
- 支持静默卸载

### 3. 其他优化
- 移除系统健康评分功能
- 优化软件卸载准确性
- 修复多个已知问题

## 🔧 技术改进

### 1. 软件管理准确性提升
- 使用 PowerShell 从注册表获取软件信息（最准确）
- 使用 WMI 补充获取软件信息
- 双重验证确保准确性

### 2. 代码优化
- 移除不必要的功能代码
- 优化错误处理
- 改进日志记录

## 📦 安装说明

### 系统要求
- Windows 10/11 64位
- 4GB RAM 以上
- 100MB 可用磁盘空间

### 安装步骤
1. 下载 `pc-toolbox-v1.5.0-portable.zip`
2. 解压到任意目录
3. 运行 `pc-toolbox.exe`

## 📖 主要功能

### 1. 系统信息
- 查看详细的系统信息
- 查看硬件信息
- 导出系统报告

### 2. 软件管理
- 查看已安装的软件列表
- 卸载软件
- 批量卸载软件
- 导出软件列表

### 3. 系统优化
- 磁盘清理
- 注册表清理
- 系统服务优化
- 启动项管理
- 浏览器数据管理

### 4. 系统工具
- 进程管理器
- 温度监控
- 浏览器扩展管理
- 导出系统报告

## 🗑️ 移除的功能

- 系统监控仪表板（不准确）
- 系统健康评分功能
