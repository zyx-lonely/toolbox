# PC Toolbox 用户手册

## 📘 简介

PC Toolbox 是一款功能强大的 Windows 系统优化和管理工具，集成了系统信息查看、系统监控、进程管理、软件管理、启动项管理、系统优化等多种功能。

## 🚀 快速开始

### 启动应用
- 从开始菜单或桌面快捷方式启动 PC Toolbox
- 首次启动可能需要几秒钟加载

### 主界面介绍
- **左侧菜单**: 功能导航菜单
- **右侧内容区**: 功能页面
- **全局搜索**: 按 Ctrl+K 唤醒全局搜索
- **主题切换**: 右上角月亮图标切换深色/浅色模式

### 全局搜索
- 快捷键: `Ctrl+K`
- 支持搜索功能、软件、设置等
- 使用 ↑↓ 导航，Enter 确认

## 📊 主要功能

### 1. 系统信息
**路径**: 系统信息

**功能**:
- 查看详细的系统信息（操作系统、CPU、内存、硬盘等）
- 查看硬件信息（主板、显卡、声卡等）
- 导出系统报告

**使用**:
1. 点击左侧菜单"系统信息"
2. 查看系统信息
3. 点击"导出报告"按钮，生成系统报告

### 2. 系统监控
**路径**: 系统工具 → 系统监控

**功能**:
- 实时监控 CPU 使用率
- 实时监控内存使用率
- 实时监控磁盘使用情况
- 查看运行中的进程

**使用**:
1. 点击左侧菜单"系统监控"
2. 查看实时监控系统状态

### 3. 进程管理器
**路径**: 系统工具 → 进程管理器

**功能**:
- 查看运行中的进程
- 结束进程
- 设置进程优先级
- 批量结束进程

**使用**:
1. 点击左侧菜单"进程管理器"
2. 查看进程列表
3. 选择进程，点击"结束"按钮结束进程
4. 选择多个进程，点击"批量结束"按钮批量结束进程

### 4. 软件管理
**路径**: Windows 优化 → 软件管理

**功能**:
- 查看已安装的软件列表
- 卸载软件
- 批量卸载软件
- 导出软件列表

**使用**:
1. 点击左侧菜单"软件管理"
2. 查看已安装的软件列表
3. 选择软件，点击"卸载"按钮卸载软件
4. 选择多个软件，点击"批量卸载"按钮批量卸载软件
5. 点击"导出"按钮，导出软件列表

### 5. 启动项管理
**路径**: Windows 优化 → 启动项管理

**功能**:
- 查看开机启动项
- 启用/禁用启动项
- 设置启动项延迟

**使用**:
1. 点击左侧菜单"启动项管理"
2. 查看启动项列表
3. 切换启动项的启用/禁用状态
4. 设置启动项延迟（秒）

### 6. 系统优化
**路径**: Windows 优化

**功能**:
- 磁盘清理
- 注册表清理
- 系统服务优化
- 浏览器数据管理
- 系统备份与还原

**使用**:
1. 点击左侧菜单"Windows 优化"
2. 选择需要的功能
3. 按照提示操作

## ⚙️ 设置

### 主题设置
- 点击右上角月亮图标切换深色/浅色模式
- 主题设置自动保存

### 收藏功能
- 点击页面标题旁边的星星图标收藏当前页面
- 收藏的页面会显示在全局搜索的"收藏"区域

### 最近使用
- 访问的页面会自动记录到"最近使用"
- 最近使用的页面会显示在全局搜索的"最近使用"区域

## ❓ 常见问题

### 1. 为什么软件管理打开时会弹出CMD窗口？
- 这是在读取注册表信息，正常情况下不会弹出窗口
- 如果仍然弹出窗口，请确保使用最新版本

### 2. 为什么系统监控中的数据不准确？
- 系统监控数据是通过 PowerShell 获取的，可能存在延迟
- 可以尝试刷新页面

### 3. 为什么无法卸载某些软件？
- 某些软件需要管理员权限才能卸载
- 某些软件可能没有正确的卸载信息
- 可以尝试使用管理员权限运行 PC Toolbox
