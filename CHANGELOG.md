# 更新日志 (Changelog)

## [1.4.0] - 2026-06-22

### 新增功能
- ✨ **批量操作功能**
  - 进程管理器：添加批量选择功能，支持批量结束进程
  - 软件管理：支持批量卸载软件

- ✨ **深色模式**
  - 添加深色模式切换功能
  - 支持自动跟随系统主题

- ✨ **系统健康评分**
  - 综合评估系统状态（CPU、内存、磁盘、启动项）
  - 显示健康评分、详细评分、健康问题和优化建议

- ✨ **软件管理**
  - 查看已安装的软件列表
  - 支持批量卸载
  - 导出软件列表

### 优化改进
- 🚀 **性能优化**
  - 系统监控仪表板
  - 虚拟滚动（大数据列表）
  - 全局搜索增强

- 🎨 **用户体验**
  - 全局搜索快捷键说明（Ctrl+K）
  - 最近使用工具记录
  - 工具收藏功能

### 修复问题
- 🐛 修复了多处编译错误
- 🐛 修复了结构体字段名不匹配的问题
- 🐛 修复了前端导入重复的问题

### 技术细节
- 使用 Go + Wails v2 + Vue 3 框架
- 采用 Naive UI 组件库
- 支持 Windows 7+ 操作系统
